Free Games Bot

The Free Games Bot is easy to setup and will automatically send a message in your server when a new game is free.

Commands:

Command What does it do?
FG help The bot will repond with all the commands
FG setup Setup the notifications channel
FG stop Stop the bot from sending notifications
FG get The bot will responds with the current free games
FG config Toggle the stores and change the language of the bot
FG role @role/off Assign a role for the bot to mention
